## Facility

- [x] Login
- [x] Register
- [x] Logout
- [ ] Forgot Password
- [ ] Update Profile
- [ ] Update Password
## Installation

```bash
npm install
```
## Environment Variables

To run this project, you will need to copy the .env.example file and rename it to .env and fill the variables with your own values.

`MONGODB_URL` <br>
`MONGODB_USER` <br>
`MONGODB_PASSWORD` <br>

`JWT_SECRET`<br>
## Usage

```bash
node app
```
